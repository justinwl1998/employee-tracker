# Employee Tracker

A program that keeps track of employee records.

## Description

A Node.js command line program that interacts with a SQL database to view, add, change or delete departments, roles or employees.

## Getting Started

### Dependencies

* Node.js
* npm
* Inquirer v8.2.4
* MySQL
